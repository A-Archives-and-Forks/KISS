Abra aplicaciones desde la pantalla de inicio escribiendo unas pocas letras.

La aplicación se adapta a tus hábitos para que encuentres lo que buscas aún más rápido.
Tus datos nunca saldrán de tu dispositivo y ahorrarás batería.
Tu dispositivo será más rápido, comparado con otros complejos y sobrecargados lanzadores.
KISS ocupa solo 250 KB y nunca se conecta a Internet.

Esta aplicación utiliza la API de accesibilidad de Android para habilitar la función "doble toque para bloquear la pantalla". No se accede ni almacena ningún dato en el proceso.
